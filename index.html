Experiencia Laboral

  • Bechtle AG, Alemania Data Scientist

    Sep 2023 - Abr 2024

    Esta experiencia laboral consistió en determinar las emisiones de CO2 de todos los paquetes distribuidos por la empresa, utilizando la data comoherramienta principal.

    Primero, identifiqué los diferentes softwares utilizados en el proceso logístico, con el objetivo de consolidar la información relevante de cada paquete y extraer los parámetros más importantes junto a SQL.

    A partir de esto, por medio de Python desarrollé un modelo que evaluaba la calidad de los datos entre estos tres sistemas, ya que a menudo había inconsistencias entre ellos.

    Posteriormente, por medio de Python creé un segundo modelo para calcular las emisiones de CO2, una vez que los datos de las tres bases se encontraban depurados.

    Finalmente, visualicé los resultados utilizando SQL y Power BI, generando un diagrama interactivo que permitía ver las emisiones de CO2 categorizadas por paquete.
